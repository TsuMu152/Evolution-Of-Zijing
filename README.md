# 开发者指南

## 如何构建

如果你在使用Windows，请在项目根目录执行以下命令：

```
gradlew build
```

如果你使用的是Linux/Unix：

```
./gradlew build
```

构建完成后，你可以在`build/libs`目录找到生成的归档文件。
